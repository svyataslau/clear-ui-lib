# @clear/ui Demo

Интерактивная демонстрация всех компонентов библиотеки @clear/ui.

## 🚀 Запуск демо

### Способ 1: Через npm скрипт
```bash
npm run demo
```

### Способ 2: Прямой запуск сервера
```bash
node demo/server.js
```

### Способ 3: Открыть файл напрямую
Просто откройте `demo/index.html` в браузере (но некоторые функции могут не работать из-за CORS).

## 🌐 Доступ

После запуска сервера откройте браузер и перейдите по адресу:
```
http://localhost:3000
```

## 📱 Что включено в демо

### Компоненты
- **Button** - Все варианты, размеры и состояния
- **Input** - Различные типы и состояния
- **Card** - Варианты с разными стилями
- **Avatar** - Размеры и fallback варианты
- **Typography** - Все варианты типографики

### Интерактивные элементы
- Рабочие формы с валидацией
- Интерактивные кнопки с событиями
- Динамическое обновление контента
- Адаптивный дизайн

## 🎨 Особенности

- **Полная функциональность** - Все компоненты работают как в реальном приложении
- **Адаптивный дизайн** - Работает на всех устройствах
- **Tailwind CSS** - Использует ту же дизайн-систему
- **React 18** - Современная версия React
- **Babel** - Поддержка JSX и современных возможностей

## 🔧 Технические детали

Демо использует:
- React 18 (CDN)
- React DOM 18 (CDN)
- Babel Standalone (CDN)
- Tailwind CSS (CDN)
- Простой HTTP сервер на Node.js

## 🛠️ Разработка

Для разработки с реальной библиотекой:

1. Запустите сборку в режиме разработки:
   ```bash
   npm run dev
   ```

2. Запустите демо сервер:
   ```bash
   npm run demo
   ```

3. Откройте http://localhost:3000

Теперь любые изменения в компонентах будут автоматически отражаться в демо!
